# Task Manager API

**Task Manager API** — это REST API для управления списком задач с поддержкой **JWT-авторизации**. Каждая задача привязана к конкретному пользователю, который может зарегистрироваться, войти и управлять своими задачами.

## Стек:
- **Python 3.10+**
- **Flask**
- **Flask-RESTX** — для создания REST API и документации Swagger.
- **Flask-JWT-Extended** — для JWT-авторизации.
- **SQLite** — для хранения данных.
- **SQLAlchemy** — ORM для работы с базой данных.
